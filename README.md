# port-scanner-project
A web-based port scanner built using Flask that allows users to scan open ports and identify running services.

##  Features
- Scan open ports of any target
- Detect common services (HTTP, SSH, etc.)
- Fast multithreaded scanning
- Clean UI with results table

---

##  Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
